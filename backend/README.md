# FastAPI Backend

## Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start development server:

```bash
uvicorn app.main:app --reload --port 8000
```

## Endpoints

- `GET /api/v1/`
- `GET /api/v1/health`

## Environment

Copy `.env.example` to `.env` and adjust values if needed.
